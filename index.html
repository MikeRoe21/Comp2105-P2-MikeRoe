HTML

HTML Stands for Hypertext Markup Language. It is the building block for all that we do in web design. HTML can be thought of as the "foundation" in a house structure.

CSS

CSS stands for "Cascading Style Sheets". CSS is what we use to make our HTML code display nicely and organize websites into neat little categories. CSS can be thought of as the decorations in a house structure.

JavaScript

JavaScript is the functionality of our web applications. It is what causes the sites to perform tasks, such as animations. JavaScript can be thought of as the windows and doors of a house.

Cloud Hosting

Cloud hosting is an option for designers to store their creations online. One example of cloud hosting is Microsoft's Azure. Cloud hosting is quick and easy to learn and set up so files don't clog your computer system!
